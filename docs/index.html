松野屋 巧 職務経歴書

2006年よりインフラ・運用領域に従事。近年はSREとして、AWS基盤の信頼性向上、運用自動化、 コスト最適化、障害対応の標準化を中心に、リモート主体で支援しています。

職務要約

都内案件を中心に、SRE/インフラ領域を担当しています。 現在は食品業界案件でSREリーダーとして参画し、横断タスク調整、障害対応の指揮、 ポストモーテム運用、Auroraメジャーバージョンアップ推進を担当しています。 小売・自動車・不動産テック・金融など複数業界で、Terraformを軸とした Infrastructure as Codeの標準化、監視基盤整備、運用改善を推進してきました。

強み

  • 課題の構造化から設計・実装・運用定着まで一貫して推進（IaC整備、監視改善、運用フロー標準化）
  • 障害時の初動指揮、関係者調整、再発防止策の実装までを含めたインシデントマネジメント
  • 大規模Terraform環境の分割・再設計・モジュール化による変更競合の抑制と生産性向上
  • RI/Savings Plansやスポット活用を含むコスト最適化施策の立案・実行
  • リモート主体の複数チーム連携における、合意形成とドキュメントベース運用
